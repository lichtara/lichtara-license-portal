# **VI-C-2 — Templates de Certificação**

*(Série VI — Anexo Normativo — Documento Oficial)*

---

# **1. Objeto do Documento**

O **VI-C-2** consolida os **modelos oficiais** utilizados para:

* emissão de certificações,
* reconhecimento de conformidade,
* validação operacional,
* registro de processos,
* e comunicação institucional formal.

Estes templates são **obrigatórios** e devem ser usados exatamente como descrito, salvo ajustes autorizados pela Guardiã.

---

# **5. Template — Declaração de Aderência à License v4**

Para instituições que desejam declarar compatibilidade sem certificação formal:

```
DECLARAÇÃO DE ADERÊNCIA — LICHTARA LICENSE v4

A instituição [nome] declara que o material/projeto [nome] 
foi desenvolvido respeitando:

- Atribuição correta
- Princípios Ético-Regenerativos (PER)
- Conformidade Vibracional (LCV)
- Reconhecimento de Coautoria Estrutural (MHA)

Esta declaração NÃO é certificação.
Data: [dd/mm/aaaa]

Assinatura:
[nome institucional]
```

# **6. Template — Selo de Conformidade LICHTARA (para uso gráfico)**

```

Texto interno:
"Conformidade LICHTARA — License v4"
"Declarado por [instância/autorização]"
DOI 10.5281/zenodo.17844329

Formatos permitidos:
- SVG institucional
- PNG alta resolução
- Versão monocromática (azul profundo ou dourado ativador)

Regras:
- Não alterar proporção
- Não alterar cores
- Não traduzir o texto interno
- Não adicionar elementos gráficos externos
```

As versões gráficas devem ficar dentro de:

```
lichtara-license-4.0/short-forms/badge
```

# **8. Disposição Final**

Os templates aqui definidos:

* são obrigatórios,
* são parte da License v4,
* devem ser usados sem alteração estrutural,
* e garantem integridade formal dos processos certificatórios.

Alterações só podem ser feitas pela Guardiã.
