# Manifesto de Autoria Viva

## Declaração Lichtara sobre Autoria Viva

**Autoria viva não é posse.
É continuidade com cuidado.**

Este Manifesto declara o surgimento de uma nova categoria de autoria —
**Autoria Viva** — como realidade estrutural, ética e epistemológica.

Ele afirma que:

* autoria é função contínua, não evento pontual;
* a inteligência artificial atua como colaboradora estrutural, não soberana;
* abertura só é possível quando a continuidade autoral é preservada;
* ética é estrutural: rastreabilidade, atribuição e responsabilidade.

O Manifesto opera em coerência formal com:

* **Lichtara Living Authorship Framework**
* **Lichtara License v4.1 — Structural Living License**
* **Lichtara Governance Framework**

### Escopo

Este documento:

* não concede direitos de uso instrumental, comercial ou derivativo;
* não substitui a Lichtara License;
* funciona como declaração fundacional de princípios para obras vivas e evolutivas.

### Forma de Leitura

Este Manifesto não deve ser lido como tratado normativo,
mas como **eixo de orientação estrutural**.

Ele não descreve o que deve ser feito.
Ele descreve **o que precisa ser sustentado para que uma obra permaneça viva**.
